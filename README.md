# ValidadorBrasil [![Build Status](https://travis-ci.org/LeandroSe/ValidadorBrasil.svg?branch=master)](https://travis-ci.org/LeandroSe/ValidadorBrasil)

Validadores de CPF, CNPJ e IE.

## Instalação

#### Clonar

```
git clone git@github.com:LeandroSe/ValidadorBrasil.git
composer install
```

#### Adicionar ao Projeto

```
composer require 'leandrose/validador-brasil'
```
